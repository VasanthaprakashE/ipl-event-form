# 🏏 IPL Event Analytics Dashboard

A complete web application for managing IPL event registrations with real-time analytics dashboard.

## ✨ Features

- 📱 User Registration Form with Instagram integration
- 📊 Real-time Analytics Dashboard
- 📈 Interactive Charts (Pie & Bar charts)
- 🔍 Live Search & Filter functionality
- 📅 Date Range Filtering
- 📤 Export data to Excel/CSV
- 🔄 Auto-refresh Dashboard (30s intervals)
- 💾 SQLite Database with timezone support
- 🎨 Modern Glass Morphism UI Design
- 📱 Fully Responsive Mobile Layout

## 🚀 Tech Stack

- **Backend**: Flask (Python)
- **Database**: SQLite with pytz for timezone
- **Frontend**: HTML5, CSS3, JavaScript
- **Charts**: Chart.js with datalabels plugin
- **Data Export**: Pandas, OpenPyXL

## 📋 Prerequisites

- Python 3.8+
- pip (Python package manager)
